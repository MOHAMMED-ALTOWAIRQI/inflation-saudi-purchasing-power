# Impact of Inflation on Purchasing Power in Saudi Arabia

## Overview
This project analyzes how inflation affects the purchasing power of individuals in Saudi Arabia using statistical methods and real-world data.

## Objective
To study the relationship between inflation and consumer behavior, and understand how rising prices impact different socioeconomic groups.

## Methodology
- Data collected from the Saudi General Authority for Statistics (GASTAT)
- Descriptive statistical analysis
- Regression analysis
- Chi-square tests

## Key Findings
- Inflation significantly affects lower-income groups
- Many individuals adjust spending habits or use savings
- Weak correlation found between some variables, suggesting external influences

## Data Analysis
- Time series analysis of inflation rates (2016–2024)
- Histogram analysis of income and expenses
- Pie charts for savings and purchasing power impact
- Relationship analysis between variables
